# Kaushik Portfolio

A modern responsive developer portfolio built with HTML, CSS and JavaScript.

## Features

- Dark Theme
- Responsive Design
- Animated Hero
- Typing Animation
- Particle Background
- Glassmorphism
- Scroll Animations
- Smooth Navigation
- Scroll Progress Bar

## Technologies

- HTML5
- CSS3
- JavaScript
- Typed.js
- AOS

## Run

Open `index.html` in your browser.
